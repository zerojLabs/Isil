Formatos de parrafo y utilidad.

* Puedes usar la etiqueta strong para dar enfasis.
* Puedes usar la etiqueta mark para resaltar texto.
* Puedes usar la etiqueta em para dar enfasis al texto en italica .
* Puedes usar el elemento span para darle estilos a partes especificas de un texto

Html y CSS proporcionan diversas formas de estructurar y dar estilos a las webs de todo el mundo :D
