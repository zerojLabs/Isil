Lenguajes de programacion populares



JavaScript

Lenguaje de scripting, interpretado, que se utiliza para agregar funcionalidad a las páginas web.

Es un lenguaje de propósito general, pero se utiliza principalmente para el desarrollo web.

Es un lenguaje dinámico, lo que significa que no necesita ser compilado antes de ejecutarse.

Python

Lenguaje de propósito general, interpretado, con sintaxis simple y fácil de aprender.

Es versátil y se usa en desarrollo web, ciencia de datos hasta inteligencia artificial

Tiene una enorme cantidad de bibliotecas y marcos de trabajo

Java

Lenguaje de propósito general, compilado, con una sintaxis similar al inglés

Se utiliza para desarrollar aplicaciones para la web, móviles y escritorio.

Es multiplataforma, su código se puede ejecutar en cualquier dispositivo que tenga un intérprete de Java.
