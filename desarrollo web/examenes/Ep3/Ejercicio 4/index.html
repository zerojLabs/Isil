HTML

El lenguaje de marcas de hypertexto o HTML es el código que se utiliza para estructurar y desplegar una página web y sus contenidos como los párrafos, una lista con viñetas o imágenes y tablas de datos.

Elementos Descripción
Elemento Principal
html Es el contenedor para todos los elementos de un documento
Metadatos del documento
head Contiene el título e información relacional acerca del documento
title Provee un título para el documento
Secciones
body Es la sección donde se encuentra el contenido del documento
h1 - h6 Contiene el título e información relacional acerca del documento
Párrafos
p Representa un párrafo
