Lenguajes de Programación

Explora los lenguajes más populares y sus características.

Python | JavaScript | Java | C# | PHP | C++

Python

Python es un lenguaje de programación de alto nivel conocido por su sintaxis clara y legible.

  • Fecha de Creación: 1991
  • Paradigma: Multiparadigma
  • Frameworks Populares: Django, Flask

Usos Comunes

Desarrollo web, inteligencia artificial, análisis de datos.

Ejemplo


print("Hola, Mundo!")

Java

Java es un lenguaje de programación de propósito general y orientado a objetos.

  • Fecha de Creación: 1995
  • Paradigma: Orientado a Objetos
  • Frameworks Populares: Spring, Hibernate

Usos Comunes

Desarrollo de aplicaciones empresariales, desarrollo Android.

Ejemplo


public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola, Mundo!");
    }
}

C#

C# (C Sharp) es un lenguaje de programación desarrollado por Microsoft.

  • Fecha de Creación: 2000
  • Paradigma: Orientado a Objetos
  • Frameworks Populares: ASP.NET, Xamarin

Usos Comunes

Desarrollo de aplicaciones Windows, desarrollo web con ASP.NET.

Ejemplo


using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hola, Mundo!");
    }
}

C++

C++ es un lenguaje de programación de propósito general, derivado del lenguaje C.

  • Fecha de Creación: 1979
  • Paradigma: Multiparadigma
  • Frameworks Populares: Qt

Usos Comunes

Desarrollo de software de sistema, desarrollo de juegos.

Ejemplo


#include 

int main() {
    std::cout << "Hola, Mundo!" << std::endl;
    return 0;
}

JavaScript

JavaScript es un lenguaje de programación ampliamente utilizado para desarrollo web.

  • Fecha de Creación: 1995
  • Paradigma: Multiparadigma
  • Frameworks Populares: React, Angular, Vue.js

Usos Comunes

Desarrollo web interactivo, construcción de interfaces de usuario.

Ejemplo


const saludar = () => {
    console.log("Hola, Mundo!");
};
saludar();

Kotlin

Kotlin es un lenguaje de programación moderno desarrollado por JetBrains.

  • Fecha de Creación: 2011
  • Paradigma: Orientado a Objetos, Funcional
  • Frameworks Populares: Ktor (para desarrollo web)

Usos Comunes

Desarrollo de aplicaciones Android, desarrollo web con Ktor.

Ejemplo


fun main() {
    println("Hola, Mundo!")
}
