Introducción

C++ es un lenguaje de programación de propósito general que extiende el lenguaje de programación C con características de programación orientada a objetos.

En esta sección, exploraremos los principios básicos de C++ y su aplicación en el desarrollo de software.

Características

Programación Orientada a Objetos

C++ soporta programación orientada a objetos, lo que permite organizar el código en clases y objetos para facilitar la reutilización y la estructuración del código.

Memoria Dinámica

C++ permite la asignación y liberación de memoria de forma manual, ofreciendo un mayor control sobre los recursos del sistema.

Plantillas (Templates)

C++ introduce el concepto de plantillas que permite la creación de código genérico para trabajar con diferentes tipos de datos sin perder eficiencia en tiempo de ejecución.

Hola, Mundo!

Clases y Herencia


class Animal {
public:
    virtual void hacerSonido() const = 0;
};

class Perro : public Animal {
public:
    void hacerSonido() const override {
        std::cout << "Guau guau" << std::endl;
    }
};

Frameworks

Si bien C++ no tiene frameworks integrados como otros lenguajes, es común utilizar bibliotecas adicionales para tareas específicas, como SDL para gráficos y OpenGL para gráficos 3D.
