Estructuras de Control


Permiten a los programadores tomar decisiones y repetir tareas. Las dos principales categorías de estructuras de control son las condicionales y los bucles.





If - Else

La estructura condicional if permite ejecutar un bloque de código si una condición es verdadera.


If - Elif - Else

La cláusula elif (else if) permite evaluar condiciones adicionales y una cláusula else para manejar el caso en que ninguna de las condiciones anteriores sea verdadera.

Bucles


Estructuras de control de flujo en programación que permiten ejecutar un bloque de código de manera repetida.




While

Se utiliza cuando no se conoce de antemano cuántas iteraciones serán necesarias y se ejecutará mientras una condición sea verdadera.

For

Se utiliza cuando se conoce la cantidad exacta de iteraciones que se deben realizar.

Break

Se utiliza para salir inmediatamente de un bucle.

Cuando se encuentra la palabra clave break dentro de un bucle (for o while), el bucle se interrumpe y el control del programa pasa a la siguiente instrucción después del bucle.

Continue

Se utiliza para pasar a la siguiente iteración del bucle sin ejecutar el resto del código dentro del bucle.

Cuando se encuentra la palabra clave continue, el control del programa vuelve al principio del bucle para la siguiente iteración

Funciones


Es un bloque de código reutilizable que realiza una tarea específica. Las funciones permiten organizar y estructurar el código al dividirlo en piezas más pequeñas y manejables. Esto facilita la comprensión del código, promueve la reutilización y facilita el mantenimiento.




Definición y Llamada

Para definir una función en Python, se utiliza la palabra clave def, seguida del nombre de la función y paréntesis que pueden contener parámetros.

Para ejecutar o "llamar" una función, simplemente se utiliza su nombre seguido de paréntesis.

Parametros y Argumentos

Las funciones pueden aceptar parámetros, que son valores que la función utiliza durante su ejecución.

Los argumentos son los valores reales que se pasan a la función cuando se llama.

Valor de Retorno

Una función puede devolver un valor utilizando la palabra clave return.

Esto permite que el resultado de la función se utilice en otras partes del programa.

Ámbito de Variables

Las variables definidas dentro de una función tienen un ámbito local.

Lo que significa que solo son accesibles dentro de esa función. Solo las variables fuera de la función tienen un ámbito global.
