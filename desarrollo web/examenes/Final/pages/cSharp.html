Introducción

C# (pronunciado "C sharp") es un lenguaje de programación moderno desarrollado por Microsoft. Fue diseñado para ser simple, eficiente y seguro.

En esta sección, exploraremos los principios básicos de C# y su aplicación en el desarrollo de software.

Características

Orientado a Objetos

C# es un lenguaje orientado a objetos, lo que significa que se basa en la concepto de objetos, que pueden contener datos en forma de campos (atributos) y código en forma de procedimientos (métodos).

Tipado Fuerte

En C#, las variables deben declararse con un tipo específico y no se pueden cambiar después de la declaración. Esto se conoce como tipado fuerte y ayuda a prevenir errores en tiempo de ejecución.

Programación Asíncrona

C# ofrece soporte para programación asíncrona mediante la palabra clave `async` y `await`, permitiendo la ejecución de tareas sin bloquear el hilo principal.

Clase y Objeto


class Persona
{
    public string Nombre { get; set; }
    public int Edad { get; set; }
}

// Crear un objeto Persona
Persona persona1 = new Persona();
persona1.Nombre = "John Doe";
persona1.Edad = 30;

Frameworks

ASP.NET

ASP.NET es un marco de desarrollo web de C# utilizado para construir aplicaciones web modernas y escalables.

Xamarin

Xamarin permite el desarrollo de aplicaciones móviles multiplataforma utilizando C# y .NET, compartiendo código entre iOS y Android.
